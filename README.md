# Dotfiles

These are my dotfiles for i3-gaps. This theme is inspired in cyberpunk hope you like it UwU
It is mainly focused around vim so you can skip some things if you don't use vim

## Getting Started

These instructions will help you set up and use this theme

### Prerequisites

You will need the following stuff installed in order to get this work perfectly

* [Rofi](https://github.com/davatorium/rofi)
* [Polybar](https://github.com/polybar/polybar)
* [Oh My Zsh](https://github.com/ohmyzsh/ohmyzsh)
* [kitty](https://github.com/kovidgoyal/kitty)
* [dunst](https://github.com/dunst-project/dunst)
* [i3-gaps](https://github.com/Airblader/i3)
* [neofetch](https://github.com/dylanaraps/neofetch)
* [feh](https://github.com/derf/feh)
* Vim centered
    * [vim](https://github.com/vim/vim)
    * [vifm](https://github.com/vifm/vifm)
* Fonts
    * [hack](https://github.com/source-foundry/Hack)
    * [Font-Awesome-5](https://github.com/FortAwesome/Font-Awesome)
